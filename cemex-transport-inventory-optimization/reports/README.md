# Results Summary

`generate_results_summary.py` renders `results_summary.pdf` -- a one-page,
print-quality summary of the transport allocation and inventory
simulation results (trip/cost bar charts + a (Q,R) policy chart on a
synthetic demand profile). Run it locally:

```
python generate_results_summary.py
```

`results_summary.pdf` itself is uploaded manually rather than
through automation, since it's a binary file.
